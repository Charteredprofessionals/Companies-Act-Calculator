# Product Context: Companies Act 2013 Calculators

## Why This Exists

Corporate compliance under the Companies Act, 2013 involves complex calculations that are error-prone when done manually. Professionals spend significant time computing values like CSR obligations, depreciation under Schedule II, managerial remuneration caps, and buyback limits. This app eliminates that friction by providing instant, accurate calculations with statutory references.

## Problems It Solves

1. **Manual Calculation Errors**: Automated formulas reduce human error in complex statutory computations
2. **Time Savings**: Instant results vs. spreadsheet-based manual calculations
3. **Compliance Awareness**: Real-time compliance status indicators highlight issues before they become problems
4. **Reference Accessibility**: Each calculator shows relevant statutory sections for easy verification
5. **Search & Discovery**: Searchable dashboard lets users quickly find the calculator they need

## How It Works (User Flow)

1. User lands on the dashboard with all calculators displayed as cards
2. User searches or scrolls to find the relevant calculator
3. User clicks a card to navigate to the calculator page
4. User inputs company-specific financial data
5. App computes results instantly (client-side)
6. Results display with compliance status, key values, and statutory references
7. User can navigate back to dashboard for other calculations

## Key User Experience Goals

- **Instant Results**: No page reloads or API calls for calculations
- **Clear Compliance Status**: Green/red indicators for compliant/non-compliant
- **Professional Feel**: Dark theme, clean typography, minimal visual clutter
- **Mobile Friendly**: Responsive design works on tablets and phones
- **Statutory Context**: Every result includes section references from the Act

## Design Philosophy

- Dark theme (neutral-950 background) for professional appearance
- Blue accent color (blue-600) for interactive elements and highlights
- Green for compliant/success states, Red for non-compliant/warning states
- Monospace font for section references (lawyer-friendly)
- Compact, information-dense layout suitable for professional use
